# Oak & Ember Family Living - GitHub Pages Starter

This is a static website starter tailored for GitHub Pages.

## Files
- `oefamily-com.html` - main page
- `style.css` - styling
- `script.js` - mobile navigation

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Rename `oefamily-com.html` to `index.html` before publishing, or duplicate it as `index.html`.
4. In GitHub, go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select your main branch and `/root` folder, then save.
7. Your site will publish at the GitHub Pages URL.

## Custom domain
To use `oefamily.com` later:
1. In GitHub Pages settings, add the custom domain.
2. In your domain registrar, point the domain DNS records to GitHub Pages.
3. Enable HTTPS in GitHub Pages once DNS is connected.
